## flomo_memos_sync

同步 flomo 笔记至 logseq

## 环境配置

session 必须配置
![setting](./images/setting.png)

## 使用

`/memo` 按照配置同步当前 flomo 笔记
`/memoToday` 同步今日 memo 笔记
![memo](./images/memo.png)
